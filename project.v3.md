# Project Requirements v3

## Revision from v2

Add explicit basis-check support for free abelian groups and mobile input usability improvement.

## Added / Changed Functional Requirements
1. For free abelian groups `Z^r`, users shall be able to enter candidate basis vectors.
2. The app shall determine whether the candidate vectors form a basis of `Z^r`.
3. The result shall explain the decision using rank and determinant (`|det| = 1`).
4. Input controls shall avoid forced mobile auto-zoom on focus.
