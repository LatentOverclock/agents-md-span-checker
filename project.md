# Project Requirements v3

## Base Requirements
- Frontend-only mathematical web app.
- Must run in modern desktop/mobile browsers.

## Functional Requirements
1. User can choose from many group families and operations.
2. App supports basic and special groups (including free abelian and free groups).
3. App shows whether chosen structure is a group and key algebraic properties.
4. App shows abelian/cyclic status and operation details.
5. App supports finite custom tables and validates group axioms.
6. App supports free-group word reduction for input words.
7. For free abelian groups `Z^r`, app supports candidate-basis input and basis decision.
8. Basis result for `Z^r` includes rank/determinant explanation (`|det| = 1`).
9. Input controls are mobile-usable without forced focus zoom.
10. App is fully client-side and deployable on GitHub Pages.
