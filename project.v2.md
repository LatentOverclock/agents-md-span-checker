# Project Requirements v2

## Revision from v1

Shift scope from narrow span-check style input to broader algebraic group exploration.

## Added / Changed Functional Requirements
1. The app shall support multiple named group families (including basic and special ones).
2. The app shall support selecting operation context where relevant.
3. The app shall report core properties (group validity, abelian, cyclic, order/notes).
4. The app shall include finite custom Cayley-table based validation.
5. The app shall include free-group word reduction functionality.
